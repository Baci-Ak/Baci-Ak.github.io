About Me
Get to Know Bassey Akom

Introduction

Welcome to my portfolio. I am Bassey Akom, a Data Scientist, Big Data Analyst, Data Engineer, and Statistician. With an M.Sc in Big Data Analytics, a B.Sc in Statistics, and a Data Science certification from Harvard University, I specialize in extracting meaningful insights from complex data sets to aid decision-making processes.

Professional Skills

Data Science

Expertise in statistical analysis, machine learning, and data visualization. Proficient in Python, R, and SQL. Experienced with PySpark, scikit-learn, TensorFlow, PyTorch, and advanced machine learning algorithms.

Big Data Analytics

Experience with Hadoop Ecosystem, Spark, and Apache Hive, and SQL. Skilled in using big data tools for processing and analyzing vast amounts of data efficiently.

Data Engineering

Skilled in designing and implementing ETL pipelines with Apache NiFi, and cloud platforms like Azure. Proficient in using Azure Data Factory (ADF) to orchestrate and automate data movement and transformation from various sources to Azure Storage (including Blob Storage and Data Lake Gen2), enhancing data integration and workflow efficiency.

Statistics

Strong foundation in statistical methods and applications, including hypothesis testing, regression analysis, and experimental design. Experienced with SAS.

  • Power BI: Developing real-time analytics and visual reports

Databases
